## 当前需求

- 文件模板管理用于按业务项维护人事文件模板，并解析文件中的标准填充符。
- 页面支持切换业务项，当前包含劳动合同、Offer、离职证明等人事文件场景。
- 左侧展示文件模板列表，支持搜索、新增、拖拽排序、删除。
- 右侧展示标准文件预览，支持上传或重新上传文件并解析填充符。
- 下方展示当前模板解析出的填充符，便于核对文件变量是否完整。
- 填充符列表弹窗支持按名称、编码和来源查询，并可复制填充符编码。

## 字段与操作

| 类型 | 内容 |
| --- | --- |
| 业务项 | 劳动合同、Offer、离职证明 |
| 模板字段 | 模板名称、文件名称、版本、填充符、更新时间 |
| 主操作 | 新增模板、上传并解析、重新上传并解析、查看填充符列表 |
| 模板操作 | 拖拽排序、删除 |
| 填充符操作 | 查询、按来源筛选、复制编码 |

## 业务规则

- 新增模板必须填写模板名称并上传文件。
- 上传文件后系统模拟解析标准填充符，填充符来源于当前业务项的标准字段库。
- 删除模板需要二次确认。
- 当前选中的模板决定预览内容和解析填充符列表。
- 填充符复制后需要给出轻量反馈，便于配置人员继续编辑文件。
