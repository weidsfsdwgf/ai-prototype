# 薪酬统计

## 基本信息

| 项目 | 内容 |
| --- | --- |
| 所属系统 | OA |
| 一级目录 | OA |
| 二级目录 | 薪酬管理 |
| 菜单路由 | `/oa/payroll/statistics` |
| 页面类型 | 列表页 + 归档详情 |

## 当前需求

- 在 OA 下新增二级目录“薪酬管理”，并新增页面“薪酬统计”。
- 页面用于归档每个月已完成审批的薪酬审批数据。
- 查看单个月度归档时，复用薪酬审批详情的部门视图和员工视图，保持部门树、员工表、明细弹窗等交互一致。

## 字段与操作

| 区域 | 内容 |
| --- | --- |
| 顶部统计 | 归档月份、最近月份、累计实发、月均薪酬 |
| 查询区 | 薪酬月份、审批单号、发起人、归档状态、归档时间 |
| 月度归档列表 | 薪酬月份、审批单号、组织、发起人、薪酬人数、实发合计、人均薪酬、审批通过时间、归档时间、状态、操作 |
| 归档详情 | 部门视图、员工视图、重置筛选、部门下钻、字段明细、员工薪资详情 |

## 业务规则

- 仅展示已进入归档或归档中的薪酬审批数据。
- 月度归档数据来源于对应月份薪酬审批通过后的快照。
- 归档详情默认按选中的月度归档单展示，并保留审批时的部门与员工维度。
- 归档页用于历史查询与导出，不承担薪酬审批办理动作。
