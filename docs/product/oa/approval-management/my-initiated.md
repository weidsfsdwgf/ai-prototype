## 当前需求

- 展示当前账号发起的审批。
- 支持按流程名称、单据号、审批摘要、状态、结果、发起人、创建时间查询。
- 进行中的审批可以撤回。
- 详情页展示审批详情和审批链路，不允许审批办理操作。
- 审批摘要统一使用 `{{字段名1}}：{{字段值1}} | {{字段名2}}：{{字段值2}}...` 格式。
- 详情页不单独展示审批摘要，右上角支持全屏展示。

## 查询条件

- 流程名称
- 单据号
- 审批摘要
- 状态
- 结果
- 发起人
- 创建时间

## 列表字段

| 字段 | 说明 |
| --- | --- |
| 流程名称 | 审批流程名称 |
| 单据号 | 业务单据编号 |
| 发起人 | 审批发起人 |
| 审批摘要 | 审批内容摘要 |
| 状态 | 进行中、已结束 |
| 结果 | 处理中、已通过、已驳回、已撤回 |
| 创建时间 | 审批发起时间 |
| 结束时间 | 审批结束时间 |

## 业务规则

- 状态包含进行中、已结束。
- 已结束代表审批已经被撤回、驳回或者完全审批通过。
- 结果包含处理中、已通过、已驳回、已撤回。
- 状态为进行中时可撤回；已结束不可撤回。
- 进行中单据需要同时展示详情和撤回两个操作。
